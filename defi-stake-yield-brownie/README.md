# frontend

## installation 
- npx
```sh
npm install -g npx
```

- yarn
```sh
npm install -g yarn
```
## create app ~ start
```
npx create-react-app front_end --template typescript
```
```
cd front_end
yarn start
```

## usedapp

```
yarn add @usedapp/core
```
## mui
```
yarn add @mui/material @mui/styles @emotion/react @emotion/styled
```
